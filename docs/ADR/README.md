# ADR Index

ADRs live in this folder and capture architectural decisions that are difficult to reverse.

## Existing ADRs

## How to add a new ADR
1) Copy `0000-adr-template.md` to the next number (e.g., `0002-...`).
2) Fill context, decision, consequences, and rollout.
3) Set status to `Proposed`, then update to `Accepted` when approved.
