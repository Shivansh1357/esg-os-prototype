# Security Policy (v1.0.0)

## Supported Versions
We patch the latest minor for security issues.

## Reporting a Vulnerability
- Email: security@yourdomain.example
- Or open a **Private Security Advisory** in GitHub.

Please include:
- Affected component (web/api/ai/jobs/sql)
- Reproduction steps & PoC
- Impact & severity assessment
- Your contact for follow-up

We commit to:
- Acknowledge within **48 hours**
- Provide a fix or mitigation plan within **7 days** for High/Critical

## Scope
- Application and APIs
- Infrastructure-as-code in this repo
- Excludes third-party services beyond our control (report to vendor)

## Disclosure
- Coordinated disclosure. We’ll credit reporters if desired.
